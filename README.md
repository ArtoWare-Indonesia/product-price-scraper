# product-price-scraper
Python web scraping project that extracts product names and prices from a practice e-commerce website using Requests and BeautifulSoup.

## Features

- Extract product titles
- Extract product prices
- Save data to CSV
- Simple and beginner-friendly code

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- CSV

## usage

python scraper.py

## output

products.csv 


## Installation

```bash
pip install -r requirements.txt
